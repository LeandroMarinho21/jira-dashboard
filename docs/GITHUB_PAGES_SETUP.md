# Configuração do GitHub Pages

Siga estes passos para publicar o dashboard:

1. Acesse o repositório no GitHub
2. Vá em **Settings** > **Pages**
3. Em **Source**, selecione **Deploy from a branch**
4. Em **Branch**, escolha `main` (ou a branch principal)
5. Em **Folder**, selecione **/ (root)**
6. Clique em **Save**

O site ficará disponível em:
- Projeto: `https://<seu-usuario>.github.io/<nome-do-repo>/` (redireciona para `/dashboard/`)
- User/Org: `https://<seu-usuario>.github.io/`
